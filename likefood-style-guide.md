# LikeFood Premium Artisan Editorial Style Guide

## 1. Style Overview

### Proposed Style Name
**Premium Artisan Editorial**

### Definition
This style blends the following qualities:
- **Editorial**: refined, spacious, and led by strong typography.
- **Artisan**: handcrafted, warm, textured, and carefully curated.
- **Premium**: elegant, mature, intentional, yet never ostentatious.
- **Modern Heritage**: inspired by tradition, but expressed through a modern visual language.

### Emotional Goals
The design should communicate:
- High quality
- Sophistication
- Strong aesthetic taste
- Trustworthiness
- Warmth
- Attention to detail
- Local identity without feeling outdated
- Suitability for both everyday purchase and gifting

### Style Keywords
- Elegant
- Warm
- Curated
- Artisanal
- Editorial
- Premium
- Refined
- Natural
- Tasteful

---

## 2. Brand DNA

### How the Brand Should Be Perceived
This should not feel like a mass-market food brand. Instead, it should feel like a brand that is:
- Carefully curated
- Backed by product stories
- Visually tasteful
- Thoughtful in packaging and presentation
- Aligned with a modern lifestyle
- Strongly positioned for premium food gifting

### Brand Personality
| Attribute | Description |
|---|---|
| Refined | Not flashy, but deep and intentional |
| Warm | Approachable, natural, and never cold or overly technical |
| Curated | Every product should feel selected with care |
| Premium | Elevated, but not distant |
| Modern Artisan | Traditional roots, elevated through modern presentation |

### Style Statement
> A premium food identity system rooted in modern craftsmanship, using a warm muted palette, editorial typography, spacious layouts, and richly textured imagery to create a refined, trustworthy experience suitable for both shopping and gifting.

## 3. Brand Color System

### Color Principles
The palette should follow this direction:
- **Burgundy / wine red** as the main brand color
- **Terracotta red** as a warm supporting accent
- **Deep moss green** for a natural premium feel
- **Mustard** to reinforce the artisan character
- **Charcoal** to balance the system with a modern tone
- **Soft cream / off-white** as the primary background

### Recommended Color Palette

#### Primary Brand Colors
| Color Name | Role | Hex |
|---|---|---|
| Burgundy Deep | Main brand color | `#8B1538` |
| Terracotta Red | Warm accent | `#BE3A2B` |
| Moss Green | Natural premium tone | `#1F5A43` |

#### Secondary Colors
| Color Name | Role | Hex |
|---|---|---|
| Mustard Gold | Accent, tag, detail | `#C6A64A` |
| Charcoal | Strong text, dark surfaces, stronger borders | `#2F2F2F` |
| Dust Rose | Soft background and light highlight | `#E7D9D6` |

#### Neutral Colors
| Color Name | Role | Hex |
|---|---|---|
| Ivory White | Primary background | `#FAF7F2` |
| Warm White | Secondary background | `#F6F1EB` |
| Stone Beige | Soft border / section background | `#E6DED4` |
| Taupe Gray | Secondary text | `#7A726A` |
| Soft Black | Primary text | `#1E1B18` |

### Color Usage Rules

#### Recommended Use
- Main background: `Ivory White`
- Primary text: `Soft Black`
- Primary CTA: `Burgundy Deep`
- Secondary CTA / hover: `Terracotta Red`
- Organic / specialty / handcrafted labels: `Moss Green`
- Small accents: `Mustard Gold`

#### Avoid
- Using too many strong colors on a single screen
- Extending dark backgrounds across entire pages
- Using bright retail-style red or yellow
- Oversaturated palettes that weaken the premium tone

## 4. Typography System

### Typography Goals
Typography should create two distinct layers:
- **Brand layer**: elegant, tasteful, expressive
- **Interface layer**: readable, functional, easy to use

### Suggested Font Pairings

#### Option A — Practical and easy to implement
- **Heading / Display:** Playfair Display or Cormorant Garamond
- **Body / UI:** Inter or Manrope

#### Option B — More editorial and elevated
- **Heading / Display:** Canela / Cormorant Garamond / Libre Baskerville
- **Body / UI:** Inter / Suisse Int’l style / Neue Haas Grotesk style

### Font Usage Rules
- **Serif** should be used for:
  - Hero titles
  - Section headings
  - Quotes
  - Sub-headlines
- **Sans-serif** should be used for:
  - Body text
  - Navigation
  - Buttons
  - Inputs
  - Product descriptions
  - Prices
  - Labels

### Suggested Type Scale
| Style | Size | Weight | Use Case |
|---|---:|---:|---|
| Display XL | 56–64px | 600 | Hero title |
| Display L | 44–48px | 600 | Main section title |
| H1 | 36–40px | 600 | Page title |
| H2 | 28–32px | 600 | Section heading |
| H3 | 22–24px | 600 | Card title / subsection |
| Body L | 18px | 400–500 | Intro text |
| Body M | 16px | 400 | Standard body copy |
| Body S | 14px | 400 | Meta / short description |
| Caption | 12px | 500 | Small labels |
| Button | 14–16px | 600 | CTA |

### Typography Principles
- Headings should feel airy and elegant
- Body text should feel clear, light, and readable
- Avoid using heavy font weights everywhere
- Avoid excessive all-caps
- Use italics only as a refined accent, not excessively

## 5. Imagery and Art Direction

### Image Tone
- Natural lighting
- Real material backdrops: wood, stone, linen, handmade paper
- Warm, soft, textured tones
- Clear product detail
- Clean composition without clutter
- Styled like a curated scene rather than an industrial catalog shoot

### Three Essential Image Types
| Image Type | Purpose |
|---|---|
| Product close-up | Highlights product quality and texture |
| Lifestyle food scene | Creates emotion and usage context |
| Packaging / gift set | Reinforces the premium gifting angle |

### Avoid
- Overly vibrant photography
- Busy backgrounds
- Heavy filters
- Poor cropping
- Marketplace-style or generic e-commerce compositions

---

## 6. Layout and Grid

### Layout Philosophy
The layout should be:
- Spacious
- Rhythmic
- Clearly structured
- Focused on the main content
- Free from overcrowded blocks

### Recommended Grid

#### Desktop
- 12-column grid
- Max width: `1200–1320px`
- Gutter: `24px`
- Section spacing: `72–120px`

#### Tablet
- 8-column grid
- Gutter: `20px`

#### Mobile
- 4-column grid
- Gutter: `16px`
- Outer padding: `16–20px`

### Spacing System
Use an 8-point spacing scale:
- 4
- 8
- 12
- 16
- 24
- 32
- 48
- 64
- 80
- 96

### Spacing Principles
- White space is a core design asset
- Do not try to fill every empty area
- Divide sections clearly by content purpose
- Hero areas, section headlines, and product blocks should breathe more than supporting elements

## 7. UI Component System

### 7.1 Buttons

#### Style
- Soft rounded corners: `10–14px`
- Height: `40–48px`
- Sans-serif font, weight 600
- Refined interaction, never overly tech-driven

#### Button Types
| Type | Color | Use |
|---|---|---|
| Primary | Burgundy background, light text | Buy now, add to cart |
| Secondary | Terracotta or Moss Green | Secondary CTA |
| Outline | Burgundy / Charcoal border | Learn more |
| Ghost | Transparent background, dark text | Minor actions |

#### Usage Rules
- Each screen should have only one strongest CTA
- Avoid placing multiple equally dominant buttons next to each other
- Hover states should be subtle, usually a slight tone or border change

### 7.2 Product Cards

#### Card Structure
- Large image on top
- Product name
- Short 1–2 line description
- Price
- Tag if needed
- Small CTA

#### Card Style
- Light background
- Subtle border or very soft shadow
- Radius: `16–20px`
- Padding: `16–20px`
- Matching rounded corners for images

#### Intended Feel
- Clean
- Appetizing
- Polished
- Premium
- Easy to scan for name and price

### 7.3 Inputs / Forms

#### Style
- Light background
- Soft stone-colored border
- Focus ring in soft burgundy or terracotta
- Taupe gray placeholder text
- Clear, readable labels

#### Intended Feel
- Friendly
- Easy to fill in
- Refined
- Never overly technical like a dashboard

### 7.4 Badges / Tags

#### Recommended Tag Groups
- Best Seller
- New
- Gift Choice
- Limited
- Handmade
- Local Specialty

#### Suggested Color Direction
- Soft Burgundy
- Soft Moss Green
- Light Mustard
- Light Dust Rose

---

## 8. Iconography and Supporting Graphics

### Icon Style
- Medium-thin line icons
- Minimal
- Slightly rounded
- Consistent stroke weight

### Supporting Graphic Elements
These can be used very sparingly:
- Thin divider lines
- Elegant separators
- Light food-inspired motifs such as seeds, leaves, or subtle organic forms
- Very soft paper or linen textures in backgrounds

### Avoid
- Overly technical icons
- Strong cartoon-like illustrations
- Dense patterns
- Highly modern gradients that clash with the brand spirit

---

## 9. Tone of Voice and Brand Copy

### Tone of Voice
- Warm
- Refined
- Gently inviting
- Confident without exaggeration
- Rich in depth, but easy to understand

### Recommended Content Focus
Copy should emphasize:
- Origin
- Flavor
- Quality
- Usage
- Gifting value

### Good Examples
- “Carefully selected from quality ingredients, suitable for everyday meals as well as gifting occasions.”
- “Rich in flavor and thoughtfully packaged, ideal for both personal enjoyment and gifting.”

### Avoid
- “Super delicious and super cheap, buy now before it’s gone.”
- “Massive promotion, unbeatable price, order right away.”

## 10. Website Application

### 10.1 Homepage

#### Recommended Sections
- A brand-led hero section
- Brand introduction
- Featured products
- Gift sets / bundles
- Ingredient or regional story
- Customer feedback
- A polished footer

#### Presentation Direction
- Plenty of white space
- Large, beautiful imagery
- Serif headlines with strong presence
- CTAs that are visible but not aggressive

### 10.2 Category Page

#### Should Be More Practical
- Clear filters
- Clear sorting
- Compact cards
- Strong price visibility
- Moderately visible badges
- Convenient purchase actions

#### Note
Keep the brand colors and typography, but optimize the layout more heavily for shopping efficiency than the homepage.

### 10.3 Product Detail Page

#### Recommended Structure
- Large image gallery
- Product name
- Price
- Short description
- Key benefits / highlights
- Ingredients / storage details
- Quantity selector
- Add to cart
- Related products

#### Principle
The upper section should be strong enough to support conversion. Storytelling should continue below.

### 10.4 Cart / Checkout

#### Rules
- Simpler than the rest of the site
- Less editorial styling
- More clarity
- Easy-to-read forms with minimal distraction

#### Priorities
- Clear product information
- Clear shipping fee
- Clear total cost
- Strong checkout CTA

---

## 11. Motion and Interaction

### Motion Style
- Moderately slow
- Soft
- Refined
- Never flashy

### Recommended Effects
- Light fade-in
- Gentle slide-up
- Very subtle hover zoom on images
- Slight button tone change on hover
- Underline animation for text links

### Avoid
- Heavy animation
- Bounce effects
- Strong parallax
- Startup-tech interaction patterns that do not fit a premium artisan food brand

---

## 12. Core Consistency Principles

1. Brand first, sales second — but never forget conversion.
2. Use few strong colors and more light backgrounds.
3. Use serif to create elegance, sans-serif to maintain usability.
4. White space is an asset.
5. Product imagery must look appetizing and authentic.
6. Buttons should be beautiful, but still action-oriented.
7. Cards should stay clean and free from heavy effects.
8. Copy should feel warm and curated.
9. Every element should feel intentionally selected.
10. Avoid generic retail language.
11. Brand-driven pages and sales-driven pages should differ in presentation intensity.
12. Premium does not mean complicated.

## 14. Implementation Summary

### Elements to Preserve Most Strongly
- Color system
- Typography
- Image tone
- Spacing
- Brand character

### Elements to Adjust in Real Product Implementation
- Make CTAs clearer
- Increase price visibility
- Make product cards more practical
- Simplify checkout further
- Make mobile layouts more compact than desktop

### Implementation Formula
**Editorial elegance** + **Artisan warmth** + **Premium restraint** + **E-commerce usability**

---

## 15. Conclusion

This style system is highly suitable for food brands focused on:
- Baked goods
- Sauces and fermented products
- Dried specialties
- Confectionery
- Nuts and seeds
- Regional specialties
- Premium food gifting

The greatest strength of this style system is its ability to create a brand image that:
- Feels distinct from generic e-commerce websites
- Has a clear aesthetic point of view
- Elevates perceived product value
- Works well for both shopping and gifting experiences

In real implementation, the editorial and premium identity should be preserved, while purchase flows should be refined to be clearer, more direct, and more conversion-friendly.
